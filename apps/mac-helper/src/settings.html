Mashi Helper

Polls the frontmost app and active tab URL every 30 seconds so Mashi can suggest S2D updates. Nothing leaves your machine except the heartbeat to your Mashi instance.

Live
Accessibility permission needed macOS hides window titles from other apps until you grant Accessibility. Click below, then enable Mashi Helper in the list.

Connection

Automation permission (browser URLs)

To read the active tab URL from Safari / Chrome / Brave / Arc, macOS will prompt the first time Mashi Helper sends an AppleScript event. If you said No by accident, re-enable Mashi Helper for each browser here.

Ignore lists

One per line. Case-insensitive substring match. These never leave your machine.

Window titles and URLs are the only data this helper captures. No keylogging, no screenshots, no clipboard. Source: apps/mac-helper in the Mashi repo.
