Mashi Activity Monitor

Tab-focus signals go to your Mashi instance so it can suggest task state changes. Nothing leaves your browser except the URL, title, and timestamp.

Base URL of your Mashi instance. Heartbeats POST to /api/activity/heartbeat.
Generate at /settings/activity with scope activity:write. Stored only in chrome.storage.local; never sent anywhere except the Mashi URL above.
One hostname per line. Suffix-match supported (e.g. starting with .). Defaults already include banks, password managers, and therapy-related domains.
What this extension captures
  • URL of the active tab
  • Title of the active tab
  • Timestamp of focus
What it never captures
  • Page content, DOM, or rendered HTML
  • Form fields, cookies, keystrokes
  • Screenshots or images

Pause anytime from the extension icon. Uninstall via chrome://extensions.
